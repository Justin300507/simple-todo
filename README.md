# Simple Todo

> Generated by **ForgeAI** — AI-powered full-stack application generator.

## Features

- Allows users to add new tasks with a title and description.
- Displays all tasks, showing their title, description, and status.
- Enables users to mark tasks as 'pending' or 'completed'.

## Quick Start

### Prerequisites

- Python 3.11+
- Node.js 18+ (for frontend)

### Backend

```bash
cd app
pip install -r requirements.txt
uvicorn main:app --reload
```

API available at: `http://localhost:8000`
Interactive docs: `http://localhost:8000/docs`

### Frontend

```bash
npm install
npm run dev
```

Frontend at: `http://localhost:5173`

### Docker

```bash
docker build -t simple_todo .
docker run -p 8000:8000 simple_todo
```
